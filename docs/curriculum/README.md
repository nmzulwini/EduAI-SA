# EduAI-SA Curriculum Framework

## Purpose

This directory contains the official curriculum structure used by EduAI-SA.

EduAI-SA follows the Department of Basic Education (DBE) CAPS curriculum.

The curriculum is organised by:

- Subject
- Grade
- Term
- Topic
- Unit
- Lesson

This structure allows the AI Tutor to teach learners exactly as the curriculum is presented in South African schools while providing personalised learning support.
