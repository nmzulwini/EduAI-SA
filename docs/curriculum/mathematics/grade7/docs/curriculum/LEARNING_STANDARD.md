# EduAI-SA Learning Standard (ELS)

## Purpose

The EduAI-SA Learning Standard (ELS) defines the structure that every lesson in the platform follows.

Every subject, grade, term, topic and lesson must comply with this standard.

---

# Lesson Structure

Every lesson contains:

1. Lesson Information
2. CAPS Learning Objectives
3. Prerequisite Knowledge
4. Key Vocabulary
5. AI Lesson
6. Visual Learning
7. Worked Examples
8. Guided Practice
9. Independent Practice
10. Challenge Questions
11. Common Misconceptions
12. Real-life Applications
13. Quiz
14. Memorandum
15. Homework
16. Teacher Notes
17. Parent Support
18. Progress Tracking
19. Mastery Check
20. Next Lesson

---

# Lesson Code

Every lesson has a unique code.

Example:

MATH-G7-T1-WN-U05

Meaning:

MATH = Mathematics

G7 = Grade 7

T1 = Term 1

WN = Whole Numbers

U05 = Unit 5

---

# AI Responsibilities

The AI Tutor must:

- Explain clearly
- Teach step-by-step
- Adapt to learner ability
- Provide hints
- Encourage independent thinking
- Give instant feedback
- Track mastery
- Recommend revision when necessary

# Progress Tracking

The AI records:

- Lessons completed
- Quiz scores
- Time spent learning
- Strengths
- Areas needing improvement
- Recommended revision

---

# EduAI-SA Principle

Every learner learns differently.

The AI adapts.

The curriculum does not.
