# EduAI-SA User Flow

## Overview

This document describes how users navigate through the EduAI-SA application.

---

## Application Flow

```
Splash Screen
      │
      ▼
Login
      │
      ▼
Choose User Role
 ┌───────────────┬───────────────┬───────────────┐
 │               │               │
 ▼               ▼               ▼

 │               │               │
 ▼               ▼               ▼

 │
 ├── AI Tutor

 └── Settings
```

---

## Learner Journey

1. Open EduAI-SA
2. Log in
3. Access Home Dashboard
4. Choose a subject or AI Tutor
5. Complete lessons and activities
6. View progress and achievements

---

## Teacher Journey

1. Log in
2. Open Teacher Dashboard
3. Generate lesson plans
4. Create worksheets
5. Create assessments
6. Monitor learner progress

---

## Parent Journey

1. Log in
2. Open Parent Dashboard
3. View learner progress
4. Check homework
5. View assessment results
6. Receive notifications
