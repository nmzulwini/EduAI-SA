# Contributing to EduAI-SA

Thank you for your interest in contributing to EduAI-SA.

Our goal is to build high-quality educational technology that benefits learners, teachers, parents, and schools.

---

## Our Principles

- Put education first.
- Write clean and readable code.
- Respect all contributors.
- Focus on solving real educational problems.
- Keep learners' data safe and secure.

---

## Branch Naming

Use descriptive branch names:

- feature/login-screen
- feature/ai-tutor
- fix/bug-description
- docs/update-readme

---

## Commit Messages

Examples:

- Add login screen
- Fix authentication bug
- Update architecture documentation
- Improve AI tutor responses

---

## Pull Requests

Before submitting a pull request:

- Test your changes.
- Update documentation if needed.
- Ensure the code follows project standards.
- Describe what changed and why.

---

## Code Style

- Write clear, maintainable code.
- Use meaningful names for variables and functions.
- Keep functions small and focused.
- Add comments where necessary.

---

## Our Mission

Every contribution should help us achieve our mission:

To give every learner a personal AI tutor and every teacher a personal AI assistant.
